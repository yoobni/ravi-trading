## 공통 환경

- 초기 자본: 10,000,000원
- 수수료: 0.05% (Upbit KRW)
- 슬리피지: 0.05%
- 일봉 진입 + 일봉 청산 (잔진동 무시)
- 자본 20% / 진입
- 동시 보유 5종 (특별 명시 없으면)
