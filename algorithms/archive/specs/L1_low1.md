# L1_low1

V1_safe 베이스에서 low threshold 0.5% → 1.0% 완화.

## 룰

```
[진입]
  - 어제 일봉 close 후 (KST 09:00) 평가 → 오늘 시초가 매수
  - 어제 low ≤ (10일 최저가 × 1.01)        ← 1% 완화
  - 어제 일봉 양봉 (close > open)
  - 동시 보유 < 5종
  - 가드 통과
  → 자본 20% 매수

[청산]
  - 매일 일봉 close 평가, 일중 변동 무시
  - 익절: +5% / 손절: -3% / 5일 강제
  - 트리거 발생 시 다음날 시초가 매도

[가드]
  - BTC 일봉 trend uptrend일 때만 매수, 아니면 즉시 청산
  - 18종 중 30%+ 동시 신저가 (10일 low 1% 이내)면 매수 X + 청산
  - 자본 -10% drawdown 도달 시 모두 청산 + 10일 휴식

[재진입]
  - 같은 종목 청산 후 24h cooldown
```
